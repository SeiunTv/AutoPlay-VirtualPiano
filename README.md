# 🎹 Roblox Piano Auto-Player

An automatic piano player for Roblox Virtual Piano — supports **Virtual Piano sheets** and **MIDI files**.

![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey)
![License](https://img.shields.io/badge/License-MIT-green)

---

## Features

- 🎵 **MIDI file playback** — drop `.mid` files in the `midi/` folder and play them directly
- 📄 **Virtual Piano sheet support** — paste VP sheets or import them from a URL
- ⏯️ **Full playback controls** — play, pause, stop with keyboard shortcuts
- 🔁 **Auto-pause** — automatically pauses when you switch away from Roblox
- 🎛️ **Speed control** — adjust tempo in real time with a slider
- 🔢 **Transposition** — shift notes up or down by semitones
- 💾 **Sheet library** — save and load your favourite sheets
- 🖥️ **Floating overlay** — mini control window that stays on top while you play

---

## Installation

Double-click **`Piano Auto-Player.exe`** — no Python or dependencies required.

> Windows 10/11 only.

---

## How to use

### Playing a MIDI file
1. Put your `.mid` files in the `midi/` folder (auto-created next to the .exe)
2. Select the file in the **Fichiers MIDI** section (double-click to load)
3. Click **Démarrer** (or press `F5`), switch to Roblox within 3 seconds

### Playing a Virtual Piano sheet
1. Paste the sheet in the **Partition** text area
2. Or import directly from a virtualpiano.net URL
3. Click **Démarrer**, switch to Roblox within 3 seconds

### Keyboard shortcuts (default)
| Action | Key |
|--------|-----|
| Start  | F5  |
| Stop   | F6  |
| Pause  | F7  |
| Speed +5 | → |
| Speed -5 | ← |

Shortcuts can be changed in the **Réglages** section — click a field and press any key.
